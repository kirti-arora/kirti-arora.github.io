List of CSS3 Properties

Animation Properties

Sr. No. Property Description
1 animation Specifies the keyframe-based animations.
2 animation-delay Specifies when the animation will start.
3 animation-direction Specifies whether the animation should play in reverse on alternate cycles or not.
4 animation-duration Specifies the number of seconds or milliseconds an animation should take to complete one cycle.
5 animation-fill-mode Specifies how a CSS animation should apply styles to its target before and after it is executing.
6 animation-iteration-count Specifies the number of times an animation cycle should be played before stopping.
7 animation-name Specifies the name of @keyframes defined animations that should be applied to the selected element.
8 animation-play-state Specifies whether the animation is running or paused.
9 animation-timing-function Specifies how a CSS animation should progress over the duration of each cycle.

Background-Properties

Sr. No. Properties Description
1 border-bottom-left-radius Defines the shape of the bottom-left border corner of an element.
2 border-bottom-right-radius Defines the shape of the bottom-right border corner of an element.
3 border-image Defines the shape of the bottom-left border corner of an element.
4 border-image-outset Specifies the amount by which the border image area extends beyond the border box.
5 border-image-repeat Specifies whether the image-border should be repeated, rounded or stretched.
6 border-image-slice Specifies the inward offsets of the image-border.
7 border-image-source Specifies the location of the image to be used as a border.
8 border-image-width Specifies the width of the image-border.
9 border-radius Defines the shape of the border corners of an element
10 border-top-left-radius Defines the shape of the top-left border corner of an element.
11 border-top-right-radius Defines the shape of the top-right border corner of an element.

Color-Properties

Sr. No. Properties Description
1 color Specify the color of the text of an element.
2 Opacity Specifies the transparency of an element.

Flexible-box-Layout

Sr. No. Properties Description
1 align-content Specifies the alignment of flexible container's items within the flex container.
2 align-items Specifies the default alignment for items within the flex container.
3 align-self Specifies the alignment for selected items within the flex container.
4 flex Specifies the components of a flexible length.
5 flex-basis Specifies the initial main size of the flex item.
6 flex-direction Specifies the direction of the flexible items.
7 flex-flow A shorthand property for the flex-direction and the flex-wrap properties.
8 flex-grow Specifies how the flex item will grow relative to the other items.
9 flex-shrink Specifies how the flex item will shrink relative to the other items inside the flex container.
10 flex-wrap Specifies whether the flexible items should wrap or not.
11 justify-content Specifies how flex items are aligned along the main axis of the flex container after any flexible lengths
12 order Specifies the order in which a flex items are displayed and laid out within a flex container.

Font Properties

Sr. No. Properties Description
1 font-size-adjust Preserves the readability of text when font fallback occurs.
2 font-stretch Selects a normal, condensed, or expanded face from a font.

Outline Properties

Sr. No. Properties Description
1 outline-offset Set the space between an outline and the border edge of an element.

Multi-column Layout Properties

Sr. No. Properties Description
1 column-count Specifies the number of columns in a multi-column element.
2 column-fill Specifies how columns will be filled.
3 column-gap Specifies the gap between the columns in a multi-column element.
4 column-rule Specifies a straight line, or "rule", to be drawn between each column in a multi-column element.
5 column-rule-color Specifies the color of the rules drawn between columns in a multi-column layout.
6 column-rule-style Specifies the style of the rule drawn between the columns in a multi-column layout.
7 column-rule-width Specifies the width of the rule drawn between the columns in a multi-column layout.
8 column-span Specifies how many columns an element spans across in a multi-column layout.
9 column-width Specifies the optimal width of the columns in a multi-column element.
10 columns A shorthand property for setting column-width and column-count properties.

Visual formatting Properties

Sr. No. Properties Description
1 overflow-x Specifies how to manage the content when it overflows the width of the element's content area.
2 overflow-y Specifies how to manage the content when it overflows the height of the element's content area.
3 resize Specifies whether or not an element is resizable by the user.
4 box-shadow Applies one or more drop-shadows to the element's box.
5 box-sizing Alter the default CSS box model.

Transition Properties

Sr. No. Properties Description
1 transition Defines the transition between two states of an element.
2 transition-delay Specifies when the transition effect will start.
3 transition-duration Specifies the number of seconds or milliseconds a transition effect should take to complete.
4 transition-property Specifies the names of the CSS properties to which a transition effect should be applied.
5 transition-timing-function Specifies the speed curve of the transition effect.

Transform Properties

Sr. No. Properties Description
1 backface-visibility Specifies whether or not the "back" side of a transformed element is visible when facing the user.
2 perspective Defines the perspective from which all child elements of the object are viewed.
3 perspective-origin Defines the origin (the vanishing point for the 3D space) for the perspective property.
4 transform-origin Defines the origin of transformation for an element.
5 transform Applies a 2D or 3D transformation to an element.
6 transform-style Specifies how nested elements are rendered in 3D space.

Text Properties

Sr. No. Properties Description
1 tab-size Specifies the length of the tab character.
2 text-align-last Specifies how the last line of a block or a line right before a forced line break is aligned when text-align is justify.
3 text-decoration-color Specifies the color of the text-decoration-line.
4 text-decoration-line Specifies what kind of line decorations are added to the element.
5 text-decoration-style Specifies the style of the lines specified by the text-decoration-line property
6 text-justify Specifies the justification method to use when the text-align property is set to justify.
7 text-overflow Specifies how the text content will be displayed, when it overflows the block containers.
8 text-shadow Applies one or more shadows to the text content of an element.
9 word-break Specifies how to break lines within words.
10 word-wrap Specifies whether to break words when the content overflows the boundaries of its container.
